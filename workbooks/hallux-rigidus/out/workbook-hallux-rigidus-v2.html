1. ВИЗНАЧЕННЯ ТА АНАТОМІЯ
КЛЮЧОВА ІДЕЯ
Hallux rigidus — повна або майже повна втрата рухливості 1-го плеснофалангового суглоба (ПФС). Hallux limitus — часткове обмеження. Нормальна дорсифлексія 1-го ПФС: 55–60°. 1-а плеснова кістка товща за 2–5, сезамоїдні кістки під головкою 1-ї плеснової працюють як надколінок для великого пальця — збільшують важіль м'язів-згиначів.

2. БІОМЕХАНІКА ХОДИ
КЛЮЧОВА ІДЕЯ
3 фази перекату стопи: контакт п'ятки (супінація) → повний контакт (пронація/амортизація) → відштовхування (дорсифлексія 1-го ПФС на 55–60°). При ригідності 1-го ПФС організм обирає латеральний обхідний маневр: навантаження переноситься на 2–5 плеснові та МФС 1-го пальця (IPJ), розвивається компенсаторна гіперсупінація → патологія колін, стегон, хребта.

3. ЕТІОЛОГІЯ
КЛЮЧОВА ІДЕЯ
Hallux rigidus — мультифакторне захворювання. Основні причини: пронована/вальгусна стопа, носіння підборів з вузьким носком, остеоартроз, подагра (кристали уратів — класика з часів Гіппократа), пряма травма, вроджена дисплазія, діабет, менопауза (гормональний вплив на хрящ).

4. ПОДОЛОГІЧНІ ОЗНАКИ («ОБЛИЧЧЯ СТОПИ»)
КЛЮЧОВА ІДЕЯ
Подолог бачить hallux rigidus раніше за лікаря — по характерному «обличчю стопи». Патогномонічна ознака: М'ЯКА ЧИСТА ШКІРА ПІД 1-м ПФС (суглоб не працює → немає навантаження → немає кератозу). Одночасно: кератоз під IPJ 1-го пальця, дифузні кератози під 2–5 головками, гіпертрофія латеральних нігтьових валиків, екстензійна контрактура IPJ, хронічний оніхолізис.

5. ДІАГНОСТИКА
КЛЮЧОВА ІДЕЯ
Три діагностичних тести: активний (пацієнт піднімає пальці стоячи), пасивний (лікар виконує дорсифлексію ПРОКСИМАЛЬНОЇ фаланги, не дистальної!), рентген. Клінічний тригер для подолога: чиста шкіра під 1-м ПФС + латеральні кератози + мозоль на IPJ = негайно перевірити рухливість 1-го ПФС.
КЛІНІЧНИЙ КЕЙС
Пацієнтка, 52 роки. Скарги: біль у колінах останні 2 роки, рецидивні мозолі під 2–3 головками, деформація нігтів 1-х пальців обох стоп. Ортопед колін — «артроз». Подолог виявив: м'яка шкіра під обома 1-ми ПФС, кератоз на IPJ, оніхолізис.
Діагноз: білатеральний hallux rigidus → компенсаторне перевантаження → гонартроз.

6. ЛІКУВАННЯ
КЛЮЧОВА ІДЕЯ
Головний принцип: рокерна ригідна підошва ОБОВ'ЯЗКОВА — вона переносить перекат з суглоба на взуття. Устілки з платформою 1-го променя, вправи для збереження залишкового ROM, фізіотерапія (УХТ, НПЗП). Категорично НЕ РОБИТИ: barefoot взуття, силіконові сепаратори, йога з дорсифлексією 1-го ПФС. Хірургія: артродез у 5–7° дорсифлексії.

7. ПРОГНОЗ ТА ЗНАЧЕННЯ ДЛЯ ПОДОЛОГА
КЛЮЧОВА ІДЕЯ
Нелікований hallux rigidus → рецидивні мозолі, нігтьові розлади, прогресуючий артроз колін/стегон/хребта. Рання подологічна детекція: перевіряти рухливість 1-го ПФС у КОЖНОГО пацієнта з кератозами або нігтьовою патологією. Реалістичне консультування: устілки + взуття значно покращать стан, але не повністю усунуть кератози при повному блокуванні суглоба.
