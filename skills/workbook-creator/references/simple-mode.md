# SIMPLE Mode — Short Course Workbook

Use this mode for single lectures, webinars, short workshops, or any content with ≤ 2 distinct topics or under ~30 minutes.

## Document Structure

```
Cover (full-page gradient background)
Content page 1 (Section 1 title + tasks)
Content page 2 (Section 2 title + tasks)
...
Content page N
Reflection page (2–3 prompts)
Final page (quote + contacts)
```

**Key differences from FULL mode:**
- **No introduction page** — content starts immediately after the cover.
- **No block title pages** — content starts immediately after the cover.
- **No block conclusions pages** — the reflection page serves this purpose.
- **Flat section numbering** — sections are numbered 1, 2, 3... (not 1.1, 1.2).
- **Fewer reflection prompts** — 2–3 instead of 4–5.

## Page Counts

- **Cover**: 1 page
- **Content pages**: 3–8 pages (minimum 3, maximum 8)
- **Reflection**: 1 page
- **Final**: 1 page
- **Total**: 6–11 pages

## Content Page Rules

Each content page should contain:
- **Section heading** (if starting a new section)
- **Key Idea** — always present at the start of a new section
- **2–3 task blocks** per page (not 4–5 as in FULL mode)
- At least 2 different task types per page

In SIMPLE mode, not every section needs all 5 content elements. Use 3–4 per section:
1. **Key Idea** (always)
2. **One structuring task** (table, checklist, or chain)
3. **One question or practical task**
4. **Notes block** (optional — include on at least 50% of content pages)

## Cover Page

Same visual style as FULL mode (dark gradient, title, subtitle), but also includes:
- No student data fields (name, date, city, experience)
- No separate introduction page needed

## Reflection Page

Simplified version with 2–3 prompts:
1. "Головне, що я дізнався/дізналась" / "Главное, что я узнал(а)" (MEDIUM, 4 lines)
2. "Що я зміню у своїй практиці" / "Что я изменю в своей практике" (MEDIUM, 4 lines)
3. "Мої перші кроки" / "Мои первые шаги" (MEDIUM, 3 lines)

No "remaining questions" or "what surprised me" prompts — keep it focused.

## Task Density

SIMPLE mode is more compact:
- Prefer MEDIUM fields over LONG where possible
- Use SHORT fields more liberally for factual recall
- SKETCH blocks should be 80–90mm (not 90–100mm as in FULL)
- Tables should have 3–5 rows (not 5–8 as in FULL)
- Checklists should have 3–4 items (not 4–6)

## When to Upgrade to FULL

If during planning you find that the source material naturally splits into 3+ distinct topics with enough depth for 4+ content pages each, suggest upgrading to FULL mode.

## Example: Webinar on Anatomical Footwear (SIMPLE)

```
Cover: "Анатомічне взуття — Робочий зошит"
Page 1: Section 1 — Що таке анатомічне взуття (key idea + comparison table + question)
Page 2: Section 2 — Критерії вибору (key idea + checklist + practical task)
Page 3: Section 3 — Клінічні випадки (key idea + chain + sketch)
Page 4: Section 4 — Рекомендації пацієнтам (key idea + table + notes)
Reflection: 3 prompts
Final: Quote + contacts
Total: 7 pages
```
