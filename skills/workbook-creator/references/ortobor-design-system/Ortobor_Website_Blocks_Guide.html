Aurora mesh gradient — рецепт
М'який фон без різких ліній. Кілька radial-gradient накладених одне на одне. opacity 10–30%, ніколи не більше 35%.
/* Темний hero block */ background: radial-gradient(ellipse at 15% 20%, rgba(0,140,137,.32) 0%, transparent 50%), radial-gradient(ellipse at 85% 75%, rgba(1,73,72,.28) 0%, transparent 45%), radial-gradient(ellipse at 50% 100%, rgba(198,238,104,.05) 0%, transparent 55%), #011E1E; /* Світлий block */ background: radial-gradient(ellipse at 5% 10%, rgba(0,140,137,.12) 0%, transparent 50%), radial-gradient(ellipse at 95% 90%, rgba(1,73,72,.10) 0%, transparent 50%), radial-gradient(ellipse at 55% 55%, rgba(198,238,238,.25) 0%, transparent 55%), #F8FDFB; /* Світла анімована версія — multiply blend для орбів */ /* orb opacity: 10-18%, mix-blend-mode: multiply */ /* Підказка: непарні orbs зліва, парні справа і знизу */
