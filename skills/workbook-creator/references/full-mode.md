# FULL Mode — Multi-Block Workbook

Use this mode for courses with 3+ distinct topics, multi-module programs, or any content longer than ~30 minutes.

## Document Structure

```
Cover (no headers/footers, full-page gradient background)
Introduction page (how to use the workbook, legend of task types)
  Block 1:
    Block title page (dark background, large block number, block name)
    Section 1.1 (content pages with tasks)
    Section 1.2
    ...
    Block conclusions page (key takeaway + application question + notes)
  Block 2: (same pattern)
  Block N: (same pattern)
Reflection page (4–5 prompts: what I learned, what I'll change, first steps, remaining questions)
Final page (motivational quote, contacts, thank-you)
```

## Page Counts

- **Cover**: 1 page
- **Introduction**: 1 page
- **Per block**: 1 title page + 2–5 content pages + 1 conclusions page = 4–7 pages
- **Reflection**: 1 page
- **Final**: 1 page
- **Total for 3 blocks**: typically 15–24 pages

## Section Content Rules

Each section in FULL mode must include all 5 content elements:

1. **Key Idea** — callout block with left accent border
2. **Structuring Task** — table, checklist, or chain
3. **Reflection Question** — open-ended, requires analysis
4. **Practical Task** — case study, algorithm, comparison
5. **Notes Block** — free-form space

## Block Title Page

Full-page dark gradient background. Contains:
- Large block number (e.g., "01") in semi-transparent white, ~64pt
- Block name in uppercase bold, ~22pt
- Short description (2–3 sentences) in regular weight

No headers or footers on block title pages.

## Block Conclusions Page

Standard page with headers/footers. Contains:
- Section heading: "Висновки по блоку N" / "Выводы по блоку N"
- Question: "What is the most important idea from this block?" (MEDIUM field)
- Practice: "How will you apply this in your work this week?" (LONG field)
- Notes block (6 lines)

## Introduction Page

Standard page with headers/footers. Contains:
- Section heading: "Як працювати з воркбуком" / "Как работать с воркбуком"
- Key idea explaining the workbook purpose
- Numbered steps (4 items): read key idea → do tasks → answer questions → take notes
- Legend of task type labels (question, checklist, chain, sketch, practice)

## Reflection Page

Standard page. Contains 4–5 reflection prompts:
1. "What new things did I learn?" (MEDIUM, 4 lines)
2. "What will I change in my practice?" (MEDIUM, 4 lines)
3. "My first steps plan" (LONG, 5 lines)
4. "Questions that remain" (MEDIUM, 3 lines)
5. (Optional) "What surprised me most?" (MEDIUM, 3 lines)

## Numbering

- Sections use hierarchical numbering: 1.1, 1.2, 2.1, 2.2, etc.
- Page numbers are sequential across the entire workbook.
- Block title pages do not have page numbers.
- Cover does not have a page number.
